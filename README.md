# Neural-Emotion

Emotion prediction and application with neural networks.

## Dependencies

* dlib
* scikit image
* nolearn
* theano
* lasagne
* matplotlib
* numpy, scipy stack

To install:

```BASH
cd Neural-Emotion
sh dependencies.sh
```
